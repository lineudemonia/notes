#### A. Supervised Learning
**定义**: 已知数据有确定对应标签，通过学习现有对应关系预测新对应关系。例如：_线性回归_

1. By category: 数据对应**类别**，如良性肿瘤或恶性肿瘤
2. By value: 数据对应**数据**，如通过房子大小预测售价

#### B. Unsupervised Learning
**定义**： 已知数据并无确认标签，需通过分析找出现有数据规律
